## Installation

```sh
git clone https://github.com/salman-abedin/faint.git && cd faint && sudo make install
```

## Update

```sh
cd faint
git pull --no-rebase && sudo make install
```

## Uninstallation

```sh
cd faint
sudo make uninstall
```
